###Descrição do jogo
O jogo **Zero ou Um** é um jogo simples e é possivel que vc jogue com seus amigos. Cada jogador digita secretamente um número: **0** ou **1**. Após todos escolherem, os valores são revelados no console junto com o jogador vencedor.  

###Como executar o projeto
1. Abra o arquivo "TCPserver".
2. Execute uma (1) vez o **TCPserver**.
3. Abra o arquivo "TCPclient".
4. Execute três (3) vezes o **TCPclient**.
5. Digite a sua escolha (1 - SIM ou 2 - NÃO).
6. O jogo já esta funcionando.

###Como jogar
- Utilize as teclas **1** e **2** para confirmar o inicio da partida e para votar se quer jogar novamente ou não. Logo em seguida, use **1** ou **0** para jogar.
- O jogo termina quando um dos jogadores digita **2 = "NÃO"** nos momentos de escolhas.
